# Документация проекта "Геометрические фигуры"

## Общее описание решения

Данный проект представляет собой библиотеку для вычисления площади и периметра различных геометрических фигур. Реализованы четыре основные фигуры: прямоугольник, квадрат, треугольник и круг.

### Структура проекта:
- `Rectangle.py` - функции для работы с прямоугольником
- `Square.py` - функции для работы с квадратом  
- `Triangle.py` - функции для работы с треугольником
- `Circle.py` - функции для работы с кругом

## Модуль Rectangle.py

### area(a, b)
Вычисляет площадь прямоугольника.

**Параметры:**
- `a` (float/int) - длина прямоугольника
- `b` (float/int) - ширина прямоугольника

**Возвращает:**
- float/int - площадь прямоугольника

**Формула:** 

S = a * b

**Пример**
```python
result = area(5, 3)
print(result)  # Вывод: 15
```

### perimetr(a, b)
Вычисляет периметр прямоугольника

**Параметры:**
- `a` (float/int) - длина прямоугольника
- `b` (float/int) - ширина прямоугольника

**Возвращает:**
- float/int - периметр прямоугольника

**Формула:** 

S = 2 * (a + b)

**Пример**
```python
result = perimetr(5, 3)
print(result)  # Вывод: 16
```

## Модуль Square.py

### area(a)
Вычисляет площадь квадрата.

**Параметры:**
- `a` (float/int) - сторона квадрата

**Возвращает:**
- float/int - площадь квадрата

**Формула:** 

S = a ^ 2

**Пример**
```python
result = area(5)
print(result)  # Вывод: 25
```

### perimetr(a)
Вычисляет периметр квадрата

**Параметры:**
- `a` (float/int) - сторона квадрата

**Возвращает:**
- float/int - периметр квадрата

**Формула:** 

S = 4 * a

**Пример**
```python
result = perimetr(5)
print(result)  # Вывод: 20
```

## Модуль Triangle.py

### area(a, h)
Вычисляет площадь треугольника.

**Параметры:**
- `a` (float/int) - сторона треугольника
- `h` (float/int) - высота треугольника

**Возвращает:**
- float/int - площадь треугольника

**Формула:** 

S = a * h / 2

**Пример**
```python
result = area(6, 4)
print(result)  # Вывод: 12
```

### perimetr(a, b, c)
Вычисляет периметр треугольника

**Параметры:**
- `a` (float/int) - сторона первой стороны
- `b` (float/int) - сторона второй стороны
- `c` (float/int) - сторона третей стороны

**Возвращает:**
- float/int - периметр треугольника

**Формула:** 

S = a + b + c

**Пример**
```python
result = perimetr(3, 4, 5)
print(result)  # Вывод: 12
```

## Модуль Circle.py

### area(a, h)
Вычисляет площадь круга.

**Параметры:**
- `r` (float/int) - радиус круга

**Возвращает:**
- float/int - площадь круга

**Формула:** 

S = π * r ^ 2

**Пример**
```python
result = area(3)
print(result)  # Вывод: 28.274333882308138
```

### perimetr(r)
Вычисляет периметр круга

**Параметры:**
- `r` (float/int) - радиус круга

**Возвращает:**
- float/int - периметр круга

**Формула:** 

S = 2 * π * r

**Пример**
```python
result = perimetr(3)
print(result)  # Вывод: 18.84955592153876
```
